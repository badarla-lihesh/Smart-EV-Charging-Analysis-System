# Smart EV Charging and Power Consumption Analysis System

This project is a simple Python-based data analysis system for Electric Vehicle (EV) charging patterns.

It reads EV charging data from a CSV file and analyzes:
- Total power consumption
- Average energy usage
- Highest consumption day
- Charging cost estimation
- Charging pattern insights

The system also provides a simple suggestion to improve energy efficiency based on usage patterns.

## Technologies Used
- Python
- Pandas
- CSV Data Handling
- VS Code

## Conclusion
This project demonstrates basic data analysis using Python and helps understand EV charging behavior in a simple and structured way.
